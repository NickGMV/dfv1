Speaker notes editor 1.0

How it works

Getting in to the editor mode

First access the speaker notes on your presentation by pressing 'S'.
In the notes section click the edit mode button to open a new editor session.
The editor window should track to wherever you are in the speaker notes, if this doesn't happen simply change 1 slide to the left or right and speaker notes will track to that location.

Once you have started the editor mode you should have a window that looks like this.

Button functions explained

There are 5 basic features you will use

  • Save and close: Saves you changes stores them in your browser memory ready to be loaded when speaker notes is re-opened
  • Reset notes: Wipes all previously edited notes on THIS presentation once you click save and close this will be permanent, to undo this (if you haven't saved yet) close down editor and speaker notes and then refresh the presentation
  • Reset reminders: Removes all reminders set on THIS presentation once you click save and close this will be permanent, to undo this (if you haven't saved yet) close down editor and speaker notes and then refresh the presentation
  • Clear presentation: Clears all additions to THIS presentation once you click save and close this will be permanent, to undo this (if you haven't saved yet) close down editor and speaker notes and then refresh the presentation
  • Tracker dots: You may have noticed there are no slide numbers in the editor window, it automatically tracks the speaker notes window and keeps the current slide at the bottom of the window.

How to edit the notes

To edit text open the editor window and select the pale blue sections you can type freely in these sections and linebreaks and formatting will be saved.

Adding a reminder

To add a reminder in your session click the add reminder button in the editor window, you will be prompted to input a text reminder for yourself
when you run the presentation your reminder will pop up on your speaker notes and will have to be dismissed before your can proceed.
These are good for key reminders, and must complete actions.

When you click the reminder button you will be prompted to add text for your reminder cancelling or leaving this box blank will not add a reminder to the presentation.

Making your edits appear in your speaker notes

To make your edits appear on your next session you must use close down the speaker notes and refresh the lesson

The skull and questionmark

Double clicking the skull will start a process to destroy all editor data saved on your browser this means all presentations! The only reason for using this would be to clear memory if it is full. Don't worry if you accidentally click this as you must confirm that you want all data deleted before it will proceed

The question mark will open up this readme page, you already worked that out

Adding speaker notes editor to GITHUB

To add to GITHUB repo take the editor dependencies and add it to the root of your repo, then add a script file with src= path to editor.js to any presentation you want the plugin to be accessible for.
